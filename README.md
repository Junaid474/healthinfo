# Health Blog Desktop App

Welcome to your new Health Blog! This software lets you manage your website easily, just like WordPress.

## How to Start

1.  **Open the App**:
    *   Double-click `npm run start:desktop` (or run it in your terminal).

2.  **First Time Setup**:
    *   **Connect to GitHub**: Copy your Repository URL (e.g., `https://github.com/yourname/healthinfo.git`) and paste it when asked.
    *   **Set Password**: Create a secure password for this app. You will need it every time you open the app.

3.  **Writing Posts**:
    *   Once logged in, you will see the **Dashboard**.
    *   Click **Posts** to add new articles.
    *   Use the visual editor to write, add images, and format your text.

4.  **Publishing**:
    *   When you are ready to update your live website, go to the top menu: **Publish > Publish to Live Site**.
    *   Wait a few minutes, and your changes will appear online!

## Menu Options

*   **View > View Live Site**: Preview your website.
*   **Settings > Change Password**: Update your app login password.
*   **Settings > Reset GitHub Connection**: Use this if you want to switch to a different website repository.
